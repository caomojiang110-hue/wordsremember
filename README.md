# 雅思单词复习小程序（本地网页版）

一个极简苹果风格的本地网页，用于录入和按遗忘曲线复习雅思单词。所有数据都保存在浏览器的 `localStorage` 中，仅对本机可见。

## 使用方式

1. 将本文件夹整体保存在你的电脑上（当前已经在 `雅思单词复习` 文件夹中）。
2. 用浏览器打开 `index.html`（双击即可，用 Edge / Chrome / Safari 等现代浏览器）。
3. 建议把该页面加入浏览器「收藏夹」或固定到任务栏，便于每天打开使用。

## 功能说明

- **首页**
  - 显示：
    - 已收录单词数
    - 今天需要复习的单词数
    - 累计背诵天数
  - **今日单词**：进入录入页面。
  - **开始复习**：进入复习页面，自动筛选到期需要复习的单词。
  - **每日签到**：点击后记录今天已学习，并更新累计天数。

- **录入今日单词**
  - 输入内容：
    - 英文单词（必填）
    - 中文释义
    - 英文解释
    - 常用短语（每行一个）
    - 例句（每行一个）
    - 背诵技巧 / 词根词源 / 联想记忆
  - 点击「保存单词」后：
    - 该单词会加入个人单词库。
    - 自动按照简单的遗忘曲线安排首次复习时间。
    - 下方可以看到最近添加的单词列表。

- **智能复习**
  - 支持 3 种复习模式：
    - 看中文写英文（看中写英）
    - 看英文写中文（看英写中）
    - 看英文解释写单词（看解释写单词）
  - 每一轮会根据「到期日」从近到远安排需要复习的单词。
  - 复习流程：
    1. 根据提示在输入框中写下你的答案；
    2. 点击「显示标准答案」对照；
    3. 选择「记住了」或「没记住」：
       - 记住：单词的阶段提升，下一次复习间隔变长；
       - 没记住：阶段降低或保持较低，下一次复习时间提前。

## 遗忘曲线安排（简化版）

- 内置复习间隔（单位：天，首次默认当天）：
  - 阶段 1：0 天（当天）
  - 阶段 2：1 天后
  - 阶段 3：2 天后
  - 阶段 4：4 天后
  - 阶段 5：7 天后
  - 阶段 6：15 天后
- 你每次点击「记住了 / 没记住」后，系统会根据当前阶段调整下次复习日期。

> 提示：如果以后你想接入在线英汉词典 API，也可以在 `app.js` 的录入逻辑中加入自动查询，然后把返回的释义、短语、例句填充到相应字段中。
